# Shopping_Application

A Shopping app made in Flutter with using Cubit and firebase..

## Features

* Add-Romove Product
* Show Categores
* Show ProductList
* Show Developer Data
* About Application
* Select by category
* Like Products
* User Profile

## Installation

### 1. Clone the repo

```
git clone https://github.com/MahmoudDabour1/Shopping-Application.git
cd shopping_application
```

### 2. Run pub get

```
flutter pub get
```
### 3. Run the app

```
flutter run
```
